#  Summary of Work Process

In this project, I implemented a simple banking system using different data structures in Java.
First, I used a LinkedList to store bank accounts and perform basic operations such as adding, displaying, and searching. Then, I extended the system by adding deposit and withdrawal functionality.
Next, I implemented a stack to manage transaction history, allowing undo operations, and a Queue to simulate bill payments and account request processing. I also worked with arrays to demonstrate fixed-size data storage.
Finally, I combined all components into a mini banking system with multiple menus (bank, ATM, admin), demonstrating how different data structures can work together in a real-world scenario.
This project helped me understand the practical use of LinkedList, Stack, Queue, and arrays in application development.
